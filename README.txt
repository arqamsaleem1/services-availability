== Description ==

This plugin allows to render a Services availability / scheduling form at the frontend. This is specific for Liquid Mobile API.

Use this shortcode to render form at the frontend.
[services-availablity-form ]


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `services-availability.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_shortcode('[services-availablity-form ]'); ?>` in your templates, or place [services-availablity-form ] in page editor.
